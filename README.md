# uThrive UI Product List Exercise

This repository contains a basic React scaffold for junior to mid-level frontend candidates.

## Included starter pieces

- A frontend-only mock product service (`src/services/productsService.js`)
- A simple product list UI (`src/App.jsx`)
- Tests for both the service and UI behavior (Vitest + React Testing Library)

## Run locally

```bash
npm install
npm run dev
```

## Validate

```bash
npm run lint
npm run test
npm run build
```
